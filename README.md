This program should track swim and rubiks cube times

Rubiks Cube times have the following data:
- Date (str)
- Location (str)
- Event (str)
- Round (int)
- Rank (int)
- Best (float)
- Average (float)
- Five Solves (list of five floats) 

Swim meet times have the following data:
- Date (str)
- Location (str)
- Stroke (str)
- Length (int)
- Time (float)
- Rank (str)
- Notes (str)
